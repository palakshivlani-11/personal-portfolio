Crypto Price Tracker

  • Tech Stack: Django,Celery,Redis,Sqlite3, Python, Bootstrap, HTML, CSS
  • Github URL: Project Link

The Crypto Price Trackers comprises Django as backend framework.The frontend is combination of Html,Css and bootstrap. Sqlite3 is used as a database for keeping a record of all the registered users and their access rights.The currencies will be tracked using celery and redis as broker and the tracking notifications will be sent through mail.

The crypto price tracker is a portal for all the crypto enthusiasts.The Users can view all the current crypto prices displayed on home page using api.The users need to register and then activate their account to log their crypto coin at which they bought and if the price changes the , they will be notified through mail to according to their requirement.
